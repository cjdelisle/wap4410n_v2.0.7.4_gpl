ステータス


[ローカルネットワーク] この画面には、使用しているアクセス ポイントおよびその LAN 接続に関する情報が表示されます。表示される内容は、IP アドレス、サブネット マスク、デフォルト ゲートウェイ、プライマリ DNS、およびセカンダリ DNS です。ページのデータを更新するには、[更新] をクリックします。
[ワイヤレス] この画面には、使用しているワイヤレス ネットワークに関する情報が表示されます。表示されるのは、現在のワイヤレス設定の要約です。ページのデータを更新するには、[更新] をクリックします。
[システムパフォーマンス] IP アドレス、MAC アドレス、接続ステータス、受信済みパケット、送信済みパケット、受信済みバイト、送信済みバイト、受信済みエラーパケット、およびドロップした受信パケットの、ワイヤレス、インターネット、および LAN 情報、またはそのいずれかの情報が表示されます。ページのデータを更新するには、[更新] をクリックします。
