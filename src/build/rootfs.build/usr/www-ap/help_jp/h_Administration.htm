[ローカルAPパスワード]

[ユーザ名]

アクセス ポイントへのアクセスに使用するユーザ名を入力します。設定を保存した後は、このユーザ名を使用してアクセス ポイントにアクセスする必要があります。

[AP パスワード]

アクセス ポイントへのアクセスに使用するパスワードを入力します。パスワードの長さは、4 ～ 63 文字にする必要があります。設定を保存した後は、このパスワードを使用してアクセス ポイントにアクセスする必要があります。

[パスワードの再入力]

確認のため、AP の新しいパスワードを再入力します。
[Webアクセス]

[Web HTTPSアクセス]

HTTPS を有効にすると、アクセス ポイントにアクセスする際のセキュリティを強化できます。有効にした場合は、アクセス ポイントへのアクセス時に「http://」ではなく「https://」を使用する必要があります。

[ワイヤレスWebアクセス]

ワイヤレス クライアントがデバイスにアクセスできるようにするには [有効] オプション ボタンを選択します。そうでない場合は、[無効] を選択します。
[リモートコンソール]

[セキュアシェル(SSH)]

[有効] に設定されている場合、アクセス ポイントでは SSH および Secure Telnet 経由のリモート アクセスだけが許可されます。
[SNMP]

Simple Network Management Protocol（SNMP）は、ネットワークの監視および管理用に広く使用されているプロトコルです。SNMP を有効にするには、[有効] をクリックします。

ID

[コンタクト先] フィールドには、AP 用の連絡先情報を入力します。[デバイス名] フィールドには、AP の名前を入力します。[ロケーション] フィールドには、AP が置かれている地域または場所を指定します。

[Getコミュニティ]

Get コミュニティの名前を入力します。

[Setコミュニティ]

Set コミュニティの名前を入力します。

[SNMPトラステッドホスト]

どの IP アドレスからでも AP にアクセスできるようにする場合は、[任意のIPアドレス] を選択します。特定の IP アドレス、または IP アドレスの範囲を指定する場合は 2 番目のオプションを選択し、フィールドに入力します。

[SNMPトラップコミュニティ]

トラップ コミュニティの名前を入力します。

[SNMPトラップ宛先]

宛先の IP アドレスを入力します。
