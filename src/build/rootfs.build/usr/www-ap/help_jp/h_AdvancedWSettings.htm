ワイヤレス詳細設定


[国/地域] ドロップダウン リストから国または地域を選択します。このフィールドには、ワイヤレス インターフェイスの操作の対象となる地域が表示されます。ここに表示されている国や地域以外の場所でのアクセス ポイントの操作が、法律で許可されない場合もあります。お住まいの国や地域が表示されない場合は、地域の関係機関に問い合わせるか、シスコの Web サイトを参照して、使用するチャネルの詳細を確認してください。
[ワールドワイドモード(802.11d)] お使いのワイヤレス ステーションがこのモードをサポートし、このモードを使用する場合は、この設定を有効にします。
[チャンネル帯域幅] ワイヤレス N 接続用に、チャネルの帯域幅を手動で選択できます。20MHz に設定した場合は、20MHz のチャネルだけが使用されます。40MHz に設定した場合、ワイヤレス N 接続では 40MHz チャネルが使用されますが、ワイヤレス B やワイヤレス G では引き続き 20MHz チャネルが使用されます。デフォルトは 20MHz です。
[ガード間隔] ワイヤレス N 接続用に、ガード間隔を手動で選択できます。[ショート(400ns)] と [ロング(800ns)] の 2 つのオプションがあります。デフォルトは [自動] です。
[CTS保護モード] Clear-To-Send（CTS）保護モードでは、アクセス ポイントがすべてのワイヤレス G 送信をキャッチする機能が強化されますが、パフォーマンスが大幅に低下します。デフォルト設定の [無効] のままにして、負荷の高い 802.11b トラフィック環境でワイヤレス G 製品がアクセス ポイントに送信できない場合に、必要に応じてアクセス ポイントがこの機能を使用できるようにします。この機能を完全に無効にする場合は、[無効] を選択します。
[ビーコン間隔] アクセス ポイントがワイヤレス ビーコンを送信する頻度を定義します。
[DTIM間隔] Delivery Traffic Indication Message（DTIM）の間隔を定義します。ネットワークで問題が発生している場合を除き、この設定は変更しないでください。
[RTSのしきい値] アクセス ポイントが Request to Send（RTS）の実装を開始するしきい値を定義します。ネットワークで問題が発生している場合を除き、この設定は変更しないでください。
[フラグメンテーションのしきい値] アクセス ポイントがパケットのフラグメントを開始するしきい値を定義します。ネットワークで問題が発生している場合を除き、この設定は変更しないでください。
[ロードバランシング] 使用率が設定済みのしきい値に達したときに、アクセス ポイントが新しいワイヤレス クライアントの関連付けを拒否できるようにします。この機能が不要な場合は [無効] を選択してください。
