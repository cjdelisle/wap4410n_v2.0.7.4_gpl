基本設定


アクセス ポイントにアクセスしたときに、最初に表示されるのが [基本設定] 画面です。この画面で一般的な設定を変更できます。ほとんどの場合は、アクセス ポイントのデフォルト値のままで十分です。

設定を構成した後、[各種管理] 画面を使用して、アクセス ポイント用の新しいパスワードを設定する必要があります。これによりセキュリティが強化され、アクセス ポイントが不正な変更から保護されます。
[デバイス設定]

[ホスト名]

このアクセス ポイントにホスト名を割り当てることができます。他のデバイスは、同じ Local Area Network（LAN; ローカル エリア ネットワーク）内でこのホスト名にアクセスできます。

[デバイス名]

アクセス ポイントに最大 39 文字で任意の名前を割り当てることができます。同じワイヤレス ネットワーク上で複数のアクセス ポイントを使用している場合は特に、一意で覚えやすい名前を付けると便利です。
[ネットワーク設定] - [IPv4]

[IP設定]

[自動コンフィギュレーション] を選択した場合、アクセス ポイントは DHCP サーバから IP アドレスを取得します。
[スタティックIPアドレス] を選択した場合は、IP アドレス、ネットワーク マスク、デフォルト ゲートウェイ、および DNS アドレスを対応するフィールドに入力してください。

[ローカルIPアドレス]、[サブネットマスク]、[デフォルトゲートウェイ]

アクセス ポイントに割り当てる、LAN 上のユーザから見たローカル IP アドレス、サブネット マスク、およびデフォルト ゲートウェイのアドレスを入力します。
[ネットワーク設定] - [IPv6]

有効にすると、IPv6 アドレスを使用してアクセス ポイントの Web GUI にアクセスできます。

[IP設定]

[自動コンフィギュレーション] を選択した場合、アクセス ポイントは DHCPv6 サーバから IPv6 アドレスを取得します。同時に [ルータアドバタイズメントを許可] も選択すると、アクセス ポイントは IPv6 アドレスの取得にステートレス自動コンフィギュレーションも利用します。
[スタティックIPアドレス] を選択した場合は、IPv6 アドレスとデフォルト ゲートウェイを対応するフィールドに入力してください。

[ローカルIPアドレス]、[デフォルトゲートウェイ]

アクセス ポイントに割り当てる、LAN 上のユーザから見たローカル IP アドレスおよびデフォルト ゲートウェイのアドレスを入力します。
