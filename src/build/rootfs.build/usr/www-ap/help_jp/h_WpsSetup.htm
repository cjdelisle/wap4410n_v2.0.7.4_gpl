Wi-Fi Protected Setup


[Wi-Fi Protected Setup]

[WPSオプション1]

管理者は [Wi-Fi Protected Setup] 画面の WPS ボタンを押して、ユーザがワイヤレス クライアントをアクセス ポイントに登録できるようにします。ユーザは、アクセス ポイントで WPS ボタンが押されるのと同時に、クライアント側で WPS ソフトウェア ボタンを押す必要があります。接続は自動的に設定されます。

[WPSオプション2]

これは、管理者がユーザのワイヤレス クライアントをアクセス ポイントに登録する際の、最も安全なオプションです。ユーザは、管理者に WPS PIN 番号を渡す必要があります。この番号はクライアントの WPS ユーティリティで確認できます。管理者は、クライアントの WPS PIN 番号を入力した後、ユーザを登録します。これにより、ユーザはアクセス ポイントに接続できます。

[WPSオプション3]

アクセス ポイントの WPS PIN 番号は、底面のラベルに記載されています。ユーザは、任意の WPS クライアント ユーティリティまたは Microsoft Vista を使用して、アクセス ポイントの WPS PIN 番号をクライアント デバイスに入力する必要があります。

WPS 機能を有効または無効にする

ここで WPS 機能を無効にすることはできません。次の CLI コマンドを使用して有効または無効にすることができます。
  • 現在の WPS 機能のステータスを表示する
    #get wps
  • WPS 機能を有効にする
    #set wps enable
  • WPS 機能を無効にする
    #set wps disable
