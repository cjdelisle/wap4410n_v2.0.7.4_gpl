詳細設定


[LANポート速度設定] 管理者がアクセス ポイントの LAN ポートを 100Mbps リンク速度に強制する場合は、これを有効にします。
 
[自動ネゴシエーション] 自動ネゴシエーションを有効または無効にします。有効にすると、[管理ポート速度] と [管理デュプレックスモード] がグレー表示になり、設定できなくなります。無効にすると、[管理ポート速度] と [管理デュプレックスモード] が通常表示になり、設定できるようになります。
 
[動作自動ネゴシエーション] イーサネット ポートの現在の自動ネゴシエーション モードを表示します。
 
[管理ポート速度] イーサネット ポートの速度の修正に使用します。[自動ネゴシエーション] が無効になっている場合のみ使用できます。
[動作ポート速度] イーサネット ポートの現在のポート速度を表示します。
 
[管理デュプレックスモード] イーサネット ポートのデュプレックス モードの修正に使用します。[自動ネゴシエーション] が無効になっている場合のみ使用できます。
[動作デュプレックスモード] イーサネット ポートの現在のデュプレックス モードを表示します。
 
[検出(Bonjour)] 管理者がアクセス ポイントを MAC OS で自動的に検出されるようにする場合は、これを有効にします。
 
[HTTPリダイレクト設定] 管理者が、新しく関連付けられたワイヤレス クライアントに対してすべての HTTP 要求を特定の URL にリダイレクトする場合は、これを有効にします。
 
[802.1Xサプリカント]

ネットワークでこの AP を操作するのに 802.1X 認証が必要な場合は、これを有効にします。

認証

  • [MACアドレスによる認証]
    認証に MAC アドレスを使用する場合は、これを選択します。
    ユーザ名とパスワードには小文字の MAC アドレスを使用します（xxxxxxxxxxxx など）。
  • [名前とパスワードによる認証]
    認証に名前とパスワードを使用する場合は、これを選択します。

名前とパスワード

  • [名前]：ログイン名を入力します。
  • [パスワード]：ログイン用のパスワードを入力します。
