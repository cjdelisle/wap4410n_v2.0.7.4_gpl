Advanced


LAN Port Speed Settings Enable this if Administrator wants the LAN port of the Access Point to be forced at 100Mbps link speed.
 
Auto Negotiation Enable or disable Auto Negotiation. If enabled, then "Administrative Port Speed" and "Administrative Duplex Mode" will become grey and cannot be configured. If disabled, "Administrative Port Speed" and "Administrative Duplex Mode" become normal and can be configured.
 
Operational Auto Negotiation Display current Auto Negotiation mode of the Ethernet Port.
 
Administrative Port Speed Used to fix the speed of the Ethernet Port. It's available only when "Auto Negotiation" is disabled.
Operational Port Speed Display current Port Speed of the Ethernet Port.
 
Administrative Duplex Mode Used to fix the duplex mode of the Ethernet Port. It's available only when "Auto Negotiation" is disabled.
Operational Duplex Mode Display current Duplex Mode of the Ethernet Port.
 
Discovery (By Bonjour) Enable this if Administrator wants the Access Point to be discovered by MAC OS automatically.
 
HTTP Redirect Settings Enable this if Administrator wants to redirect all initialed HTTP requests to a specified URL for newly associated wireless client.
 
802.1X Supplicant

Enable this if your network requires this AP to use 802.1X authentication in order to operate.

Authentication

  • Authentication via MAC Address
    Select this if you want to Use MAC Address for Authentication.
    We use lowercase MAC address for User name and Password,like xxxxxxxxxxxx.
  • Authentication via Name and Password
    Select this if you want to Use name and password for Authentication.

Name and Password

  • Name - Enter the login name.
  • Password - Enter the desired login password.
