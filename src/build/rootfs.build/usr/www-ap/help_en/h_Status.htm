Status


Local Network This screen displays information about your Access Point and its LAN connection. The Local Network information that is displayed is the IP Address, Subnet Mask, Default Gateway, Primary DNS and Secondary DNS. Click Refresh to update the data on the page.
 
Wireless This screen displays information about your wireless network. It displays a summary of your current wireless settings. Click Refresh to update the data on the page.
 
System Performance The System Performance information that is displayed is the Wireless, Internet, and/or LAN information for the IP Address, MAC Address, Connection Status, Packets Received, Packets Sent, Bytes Received, Bytes Sent, Error Packets Received, and Dropped Received Packets. Click Refresh to update the data on the page.
