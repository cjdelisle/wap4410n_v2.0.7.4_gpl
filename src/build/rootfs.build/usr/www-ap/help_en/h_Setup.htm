Basic Setup


The Setup screen is the first screen you will see when accessing the Access Point. Use this screen to alter general settings. For most users, the Access Point's default values should be satisfactory.

After you have configured these settings, you should set a new password for the Access Point (using the Administration screen). This will increase security, protecting the Access Point from unauthorized changes.

Device Setup

Host Name

You may assign a Host name to this Access Point. Other devices can access this Host Name within the same LAN.

Device Name

You may assign any name to the Access Point up to 39 characters. Unique, memorable names are helpful, especially if you are using multiple access points on the same wireless network.
 
Network Setup - IPv4

IP Settings

If you select Automatic Configuration, then Access Point will enquire IP address from a DHCP server.
If you select Static IP Address, please input the IP Address, Network Mask, Default Gateway and DNS addresses in the corresponding fields.

Local IP Address, Subnet Mask, and Default Gateway

Enter the Local IP Address, Subnet Mask, and Default Gateway address you want to assign to the Access Point, as seen by users on your local area network (LAN).
 
Network Setup - IPv6

If enabled, then can access the Access Point's web GUI via IPv6 address.

IP Settings

If you select Automatic Configuration, then Access Point will enquire IPv6 address from DHCPv6 server. If you check "Accept Router Advertisement" at same time, the Access Point also enquires IPv6 address through Stateless Auto Configuration.
If you select Static IP Address, please input the IPv6 Address and Default Gateway in the corresponding fields.

Local IP Address, and Default Gateway

Enter the Local IP Address,and Default Gateway address you want to assign to the Access Point, as seen by users on your local area network (LAN).
