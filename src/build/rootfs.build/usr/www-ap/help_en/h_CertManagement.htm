SSL Certificate Management Use the following options to import or export a certificate:

Export Certificate:
Click Export Certificate to export the SSL certificate as a file. When the File Download window appears, choose Save. Find the location where you want to save, enter a descriptive name for the file, and then click Save. You can then close the Download Complete window.

Install Certificate:
To install a certificate from your PC, click Browse. When the Choose File window appears, select the file, and then click Open. Finally, click Install Certificate.
