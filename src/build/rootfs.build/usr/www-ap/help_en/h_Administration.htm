Management


Local AP Password

User Name

Enter the User Name that you would like to use to access the Access Point. Once you save settings, this is the user name you will need to use to access the Access Point.

AP Password

Enter the Password that you would like to use to access the Access Point. Your password must between 4 and 63 characters in length. Once you save settings, this is the user name you will need to use to access the Access Point.

Re-enter to confirm

Re-enter the AP's new password to confirm it.
 
Web Access

Web HTTPS Access

To increase security when accessing the Access Point, you can enable HTTPS. Once enabled, users will need to use "https://" instead of "http://" when accessing the Access Point.

Wireless Web Access

Select Enabled radio button to allow wireless client to access the device. Otherwise, choose Disabled.
 
Remote Console

Secure Shell(SSH)

If set to Enabled, the Access Point will only allow remote access via Secure Shell and Secure Telnet.
 
SNMP

Simple Network Management Protocol (SNMP) is a popular network monitoring and management protocol.To enable SNMP, click Enabled .

Identification

In the Contact field, enter contact information for the AP. In the Device Name field, enter the name of the AP. In the Location field, specify the area or location where the AP resides.

Get Community

Enter the name of your Get community.

Set Community

Enter the name of your Set community.

SNMP Trusted Host

If you want to be able to access the AP from any IP address, select Any IP Address . If you want to specify an IP address or range of IP addresses, then select the second option and complete the fields provided.

SNMP Trap-Community

Enter the name of your Trap community.

SNMP Trap-Destination

Enter the destination IP Address.
