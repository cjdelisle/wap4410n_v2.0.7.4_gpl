Wi-Fi Protected Setup

WPS Option 1

An administrator can push the WPS button on the Wi-Fi Protected Setup screen to allow a user to register a wireless client with the Access Point. The user needs to push the WPS software button on the client side at the same time as the WPS button is pushed on the Access Point. The connection will be automatically set up.

WPS Option 2

This is the most secure option for an administrator to register a user's wireless client with the Access Point. The user needs to give the administrator their WPS PIN number, which is found in the client WPS utility. After entering the client's WPS PIN number, the administrator registers the user. The user can then connect to the Access Point.

WPS Option 3

The Access Point's WPS PIN number is on the label on its bottom panel. Using any WPS client utility or Microsoft Vista, the user needs to enter the Access Point's WPS PIN number into the client device.

Enable or disable WPS function.

WPS function cannot be disabled here. You can enable or disable it through following CLI commands:
  • Display current WPS function status.
    #get wps
  • Enable WPS function.
    #set wps enable
  • Disable WPS function.
    #set wps disable
