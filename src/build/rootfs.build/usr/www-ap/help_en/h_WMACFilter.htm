Wireless Connection Control


Wireless Network Access: You may choose to allow or prevent certain computers from accessing your network. Enter the MAC address of the computers that you would like to have controlled access. You may also select a MAC address from currently networked computer by clicking the Wireless Client List button.
 
Select SSID Select SSID to setup wireless connection control.
 
Connection Control SelectLocal,The Access Point will use the local MAC address table for Access Control.SelectRADIUS,The Access Point will use the MAC address table located on the external RADIUS server on the LAN for Access Control. If you do not want to specify any restrictions to your wireless network PCs,Select Disabled.
 
Local Connection Control

Choose prevent from accessing if you want restrict certain PCs from accessing your wireless network. Select Permit to access if you want to specify PCs that can only have access to your wireless network.

MAC 1 ~ MAC 20

You can manually enter up to 20 PCs MAC addresses. Enter the MAC address of the PCs that you want to either prevent/permit access to your wireless network.

Wireless Client List

By clicking this option, you will see a MAC table that will display the wireless PCs that are already wirelessly connected to the Access Point. You can select a PC by checking the checkbox. Click Apply to add it to the access list.

Primary/Backup Shared Secret

This is shared between the Wireless Access Point and the RADIUS Server while authenticating the supplicant.
