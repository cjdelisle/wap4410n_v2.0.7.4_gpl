Configuration Management


Save/Restore AP Configuration To save the Access Point's configuration file to your local PC:

Click "Save Configuration to File" button, and save the configuration file to a specific location on your PC.


To Restore a pre-saved configuration file to the AP:

Click Browse to find the configuration file you want to load to the Access Point. Click the "Load" button to load the configuration.
